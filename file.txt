1- Нужно зайти в свою учетную запись на GitHub, открыть нужный репозиторий , кликнуть на <Code>, скопировать SSH-ключ. В терминале на компьютере необходимо открыть ту директорию  , в которую необходимо скопировать репозиторий. Вводим команду git clone + ssh ключ

2-
 Подготавливаем репо на гитхаб 
 через терминал заходим в нужную папку
 git init инициализируем
 git add . добавить все файлы 
  „git commit -m ‚название коммита’ 
 Создаем и переключаемся на главную ветку main: „git branch -M main“
 Привязываем локальный репозиторий к удаленному: git remote add origin <SSH>
 git push -u origin main пушим

3.  git pull

4. Git status - проверка состояния файлов(наличие изменений)

Git remote -v просмотр подключенных удаленных репо и протокол их подключения
